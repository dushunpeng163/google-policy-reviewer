# 🌐 快速使用指南

## ⚡ 一键启动Web可视化界面

### **📦 第0步：安装依赖**

**在启动前，首先安装Python依赖包：**

```bash
# 🚀 一键安装 (推荐)
./install.sh

# 或 Python跨平台安装
python3 setup.py

# 或 手动安装核心包
pip install flask flask-cors flask-limiter aiohttp pyyaml requests python-dateutil cryptography
```

### **🌐 第1步：启动Web界面**

```bash
# 下载系统
cd google-policy-reviewer

# 一键启动Web界面
python3 launcher.py --mode web

# 或使用专用启动器
python3 start_web.py
```

**然后访问**: http://localhost:5000

### **方式3: API集成**

```bash
# 启动API服务器
python3 launcher.py --mode api

# 调用API
curl -X POST http://localhost:5000/api/v1/compliance/analyze \
  -H "X-API-Key: demo-key" \
  -H "Content-Type: application/json" \
  -d '{"app_profile": {...}}'
```

## 💡 为什么选择Web界面？

### **vs 命令行工具**
- ✅ **更友好** - 图形界面 vs 黑窗口
- ✅ **更直观** - 实时预览 vs 文本输出
- ✅ **更专业** - 可视化图表 vs 纯文字
- ✅ **更易分享** - 浏览器链接 vs 命令行截图

### **vs 传统合规咨询**
- ✅ **更快速** - 1分钟 vs 数周等待
- ✅ **更便宜** - 一次投入 vs 每次收费
- ✅ **更准确** - AI分析 vs 人工判断
- ✅ **更完整** - 代码模板 vs 纸面建议

### **vs 通用合规工具**
- ✅ **更专业** - 教育游戏垂直专精 vs 泛化检查
- ✅ **更实用** - 完整技术实现 vs 检查清单
- ✅ **更智能** - AI增强预测 vs 静态规则
- ✅ **更及时** - 24小时更新 vs 季度更新
